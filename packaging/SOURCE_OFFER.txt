Privacy Firewall — Source Code Availability
===========================================

Privacy Firewall is free software licensed under the GNU Affero General Public
License, version 3 (AGPL-3.0). The full licence text is installed alongside this
file as LICENSE.

This application is distributed as a binary bundle that also includes other free
software, notably PyMuPDF (AGPL-3.0), which the redaction engine uses to read and
rewrite PDF content streams.

WHERE TO GET THE SOURCE
-----------------------

The complete corresponding source code for this application, including the build
scripts used to produce this binary, is published at:

    https://github.com/SurajSongara/privacy-firewall-starter-kit

Each released binary is built from a tagged commit in that repository; the tag
matches the version shown by "Privacy Firewall --version".

The build recipe that produced this bundle lives under packaging/ in that
repository (privacy_firewall.spec, launcher.py, and the platform installer
scripts).

WRITTEN OFFER
-------------

If you cannot obtain the source from the address above, you may request a copy of
the complete corresponding source code for this binary, on a physical medium
customarily used for software interchange, for no more than the cost of
performing the distribution. Requests may be sent to the maintainers via the
issue tracker of the repository named above.
