# P009_REVIEW_UI

`privacy-firewall review file.pdf [--policy name]` — local, offline web review. Runs the pipeline, builds a ReviewPlan, serves a single-page app on 127.0.0.1 (random free port) and opens the browser. UI: page images (P008) with bbox overlays color-coded by suggested action (redact/ask/keep), click to toggle any detection, sidebar grouped by type with bulk actions ("keep all IFSC"), evidence tooltip showing reasons + confidence. "Apply" posts decisions → server writes the ReviewPlan and renders the redacted PDF, then shows the output path. Zero business logic in the UI — it only edits `decision` fields on the plan; the engine does everything else. Ship as optional extra `pip install privacy-firewall[ui]` (FastAPI + uvicorn); core install must not require them. Static assets inline/self-contained — no CDN.
